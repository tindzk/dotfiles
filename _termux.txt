Ctrl+Alt:
    ‘C’ → Create new session
    ‘R’ → Rename current session
    Down arrow (or ‘N’) → Next session
    Up arrow (or ‘P’) → Previous session
    Right arrow → Open drawer
    Left arrow → Close drawer
    ‘F’ → Toggle full screen
    ‘M’ → Show menu
    ‘U’ → Select URL
    ‘V’ → Paste
    +/- → Adjust text size
    1-9 → Go to numbered session
